Scaffold a new Python module in this project.

Ask the user for:
1. Module name (snake_case)
2. One-sentence description of what it does

Then create:
- `src/<package>/<name>.py` — with a module docstring and placeholder class or function
- `tests/test_<name>.py` — with a basic test file importing from the module and one placeholder test

After creating both files, run `make lint` to confirm they pass cleanly.
